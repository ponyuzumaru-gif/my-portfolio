================================================================
  博多駅前ルーム LP - 画像素材 配置ガイド
================================================================

このフォルダに下記の画像ファイルを配置してください。
ファイル名は必ず半角英数字・ハイフンのみ使用（スペース不可）。

----------------------------------------------------------------
【必須ファイル一覧】
----------------------------------------------------------------

1. hero-main.jpg
   ・使用箇所 : ヒーロー（ファーストビュー）メイン画像
   ・推奨内容 : 20〜30代の笑顔の女性がスマホやPCを操作しているイメージ
               （カフェ・自宅などの明るい背景が好ましい）
   ・推奨サイズ: 750×500px 以上（横長）
   ・注意    : 顔出し可能なフリー素材 or 購入済みの商用利用可画像を使用

2. flow-step01.jpg
   ・使用箇所 : 登録の流れ STEP01（LINEで相談）
   ・推奨内容 : スマホでLINEを操作している女性の手元
   ・推奨サイズ: 600×200px（横長）

3. flow-step02.jpg
   ・使用箇所 : 登録の流れ STEP02（オンライン面談）
   ・推奨内容 : ビデオ通話・オンライン面談のイメージ
               （ノートPCに向かって話す女性など）
   ・推奨サイズ: 600×200px（横長）

4. flow-step03.jpg
   ・使用箇所 : 登録の流れ STEP03（本登録・アカウント作成）
   ・推奨内容 : スマホでアプリ登録・フォーム入力をしているイメージ
   ・推奨サイズ: 600×200px（横長）

5. flow-step04.jpg
   ・使用箇所 : 登録の流れ STEP04（お仕事スタート）
   ・推奨内容 : 笑顔でスマホ・PCを操作している女性（自宅・カフェなど）
   ・推奨サイズ: 600×200px（横長）

----------------------------------------------------------------
【任意ファイル（OGP用）】
----------------------------------------------------------------

6. ogp.png
   ・使用箇所 : SNSシェア時のサムネイル画像（index.html内のOGPタグ）
   ・推奨内容 : ロゴ + キャッチコピーを組み合わせたバナー画像
   ・推奨サイズ: 1200×630px（OGP標準サイズ）

----------------------------------------------------------------
【画像が用意できるまでの応急処置】
----------------------------------------------------------------

画像がない状態でも index.html をブラウザで開くと、
img タグの代わりにブラウザデフォルトの「画像なし」表示になります。
本番公開前に必ず各画像を差し替えてください。
